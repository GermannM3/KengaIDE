# BLOCK 19 — IDE UX как у Cursor

## Цель
UX приближен к Cursor/VS Code: вкладки, split view, панель агента с timeline инструментов.

## Компоненты

### 1. Tabs (вкладки файлов)
- Список открытых файлов над редактором
- Клик по вкладке — переключение
- Кнопка закрытия на каждой вкладке
- Текущий файл визуально выделен

### 2. Split view
- Опционально: два редактора рядом (вертикальный split)
- Кнопка «Split» или горячая клавиша
- Второй редактор показывает выбранный файл (или пустой)

### 3. Agent side panel
- Правая панель уже есть — AI панель
- Добавить секцию «Tool timeline»: список tool_call → tool_result
- Сворачиваемый блок с историей шагов агента
